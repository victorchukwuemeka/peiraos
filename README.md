# PeiraOS

A marketing engine that generates short video scripts for Nigerian businesses — built to grow Peira-Services and eventually serve every SME that wants to win online.

---

## What It Does

PeiraOS takes a business type and location, and returns ready-to-post video scripts formatted for TikTok, Instagram Reels, and WhatsApp Status. Every script ends with a link back to Peira-Services.

---

## Why It Exists

Most marketing tools are built for Western businesses. Nigerian SMEs speak differently, sell differently, and hustle differently. PeiraOS understands that context — and turns it into content that converts.

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Laravel 12 |
| Database | MySQL |
| AI | Groq API — Llama 3.3 70B |
| Frontend | Blade + Tailwind CSS |
| Video creation | CapCut / Canva (manual, for now) |

---

## Roadmap

### Phase 1 — Internal Use
Use PeiraOS to market Peira-Services. Post scripts daily. Close web design clients. Prove the tool works.

### Phase 2 — Beta
Give free access to 5 Nigerian business owners. Watch how they use it. Iterate on feedback.

### Phase 3 — SaaS
Open to the public with a Paystack-powered subscription. Free tier, Starter, and Pro plans. Every Pro user gets a free website audit — the natural upsell into Peira-Services.
